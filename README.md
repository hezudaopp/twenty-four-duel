# 24点对战 (Twenty-Four Duel)

双人对战24点数学游戏，支持 iOS 和 Android 双平台。

## 游戏规则

使用 +、−、×、÷ 和括号，让给定的 4 个数字组成等于 24 的算式。两名玩家面对面竞速，谁先算出谁得分！

## 功能特点

- 🎮 双人面对面对战（分屏竖屏模式）
- 🎵 背景音乐和趣味音效
- 📊 交战记录和错题集
- ⚙️ 可配置难度、局数、限时
- 🔢 智能输入验证
- 📱 支持跳过和提前结束

## 项目结构

```
├── TwentyFourDuel.swiftpm/    # iOS 版本 (Swift + SwiftUI)
├── TwentyFourDuel-Android/    # Android 版本 (Kotlin + Jetpack Compose)
└── 24point-duel.html          # Web 原型
```

## iOS 版本

- 开发语言：Swift + SwiftUI
- 最低支持：iOS 16+
- 目标设备：iPad Pro 11寸 / iPhone

## Android 版本

- 开发语言：Kotlin + Jetpack Compose
- 最低支持：Android 8.0 (API 26)
- 构建工具：Gradle 8.5
